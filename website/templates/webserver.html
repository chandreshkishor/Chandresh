What Is Web Server ?

Web Server:  Web server is a program which processes the network requests of the users and serves them with files that create web pages. This exchange takes place using Hypertext Transfer Protocol (HTTP).

Example: Apache, Tomcat and Resin

How Web servers work?

Important points

If the requested web page at the client side is not found, then web server will sends the HTTP response:
  • Error 404 Not found.
  • When the web server searching the requested page if requested page is found then it will send to the client with an HTTP response.
  • If the client requests some other resources then web server will contact to application server and data is store for constructing the HTTP response.
